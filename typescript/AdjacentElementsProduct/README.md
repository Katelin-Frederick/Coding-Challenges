# Adjacent Elements Product

Given an array of integers, find the pair of adjacent elements that has the largest product and return that product.

###### Example 1:

```
Input: [3, 6, -2, -5, 7, 3]
Output: 21
Explanation: 7 and 3 produce the largest product
```

###### Example 2:

```
Input: [6, 8, 1, 7, 6, 5, 2]
Output: 48
```

###### Example 3:

```
Input: [9, 4, 3, 2, 8, 7, 3]
Output: 56
```
