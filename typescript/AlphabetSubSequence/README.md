# Almost Increasing Sequence

Check whether the given string is a subsequence of the plaintext alphabet.

###### Example 1:

```
Input: 'effg'
Output: false
```

###### Example 2:

```
Input: 'cdce'
Output: false
```

###### Example 3:

```
Input: 'ace'
Output: true
```

###### Example 4:

```
Input: 'bxz'
Output: true
```
