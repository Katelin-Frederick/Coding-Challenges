# All Longest Strings

Given an array of strings, return another array containing all of its longest strings.

###### Example 1:

```
Input: digits = ['aba', 'aa', 'ad', 'vcd', 'aba']
Output: ['aba', 'vcd', 'aba']
```

###### Example 2:

```
Input: digits = ['zyz', 'zz', 'zw', 'wkx', 'zyz']
Output: ['zyz', 'wkx', 'zyz']
```

###### Example 3:

```
Input: digits = ['a', 'b', 'c', 'd', 'e', 'f', 'g']
Output: ['a', 'b', 'c', 'd', 'e', 'f', 'g']
```
